# Publisher by Research and Development (R&D) Team

Centralized Hugo theme (UI layer) for Isolated Command websites.
Consumed by child sites via Hugo Modules:

```toml
[module]
  [[module.imports]]
    path = "github.com/isolatedcommand/Publisher"
```
